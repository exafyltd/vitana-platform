## What it is

The Voice Command Manual explains how users talk to Vitana. The user can ask for help, definitions, navigation, community context, health guidance, calendar changes, reminders, messages, business setup, wallet information, connected app setup, media playback, and marketplace discovery in natural language.

Vitana should understand four common command types: explain, navigate, act, and follow up. Explain means "what is this?" or "how do I do this?" Navigate means "open Health" or "where can I find events?" Act means "set a reminder", "send this message", "log water", or "add this to my cart." Follow up means the user wants Vitana to remember, continue, or check back later.

## Why it matters

Vitanaland is intentionally broad. Voice is how the system stays usable. A user should not need to memorize menus to benefit from the Vitana Index, Life Compass, Daily Diary, Find a Match, Events, Support, Calendar, Health, Business, Wallet, Connected Apps, or Media Hub.

Voice also supports the principle "less think, more do." The assistant should move from understanding to action quickly while still asking for confirmation before public, financial, health-sensitive, or irreversible steps.

## Where to find it

Use the ORB overlay from the global assistant button or use AI Assistant Chat when typing is easier. Voice navigation can open most authenticated screens directly, while overlays such as Calendar, Wallet, Profile Preview, or Master Action may open on top of the current screen.

## How to use it

1. Say the outcome, not the menu path. Example: "Find events this weekend" is better than "open community then events."
2. Ask "what is this?" when a screen, score, action, or recommendation is unclear.
3. Use "open" for navigation: "open my wallet", "open daily diary", "open connected apps."
4. Use verbs for actions: "set", "send", "log", "upload", "play", "book", "save", "share", "connect."
5. Confirm sensitive actions when Vitana asks. Messages, purchases, bookings, profile changes, data sharing, and subscription changes need a clear yes.

## Example commands

- "Help me."
- "What is the Vitana Index?"
- "Who is in the community?"
- "Where can I find Find a Match?"
- "Open My Journey."
- "Set a reminder to drink water at 3 PM."
- "Send this chat message by voice."
- "Connect my wearable."
- "Connect my music app."
- "Upload this short."
- "Talk to tech support."
