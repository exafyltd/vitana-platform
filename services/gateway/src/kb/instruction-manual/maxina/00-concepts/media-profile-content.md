## What it is

Media, Profile, and Community Content are how the user becomes visible inside Maxina. Media Hub supports shorts, videos, images, music, and podcasts. Feed supports posts and comments. Profile Edit lets the user enrich their identity, skills, interests, services, and visibility. Public Profile shows what others can see.

These surfaces support both belonging and opportunity. A useful profile improves matches. Useful media builds trust. Comments and posts create community signal. Services and creator content can lead to income.

## Why it matters

Health and economic independence are social in practice. People are more likely to act when they are seen, supported, and connected to relevant others. But visibility must be intentional. Vitana should help users publish, upload, and comment while protecting sensitive information.

## Where to find it

Use Media Hub for watch/listen/upload workflows, Feed for posts and comments, Profile Edit for enrichment, Public Profile for previewing visibility, and Sharing for campaigns or external distribution.

## How to use it

1. Enrich the profile before asking Vitana to find matches or clients.
2. Upload media only when the user owns the content or has permission to share it.
3. Use comments to participate in the community and create relationship signal.
4. Preview the public profile before sharing it widely.
5. Use Sharing and Data & Consent when content should go beyond the Maxina community.

## What you can ask Vitana

- "Upload this short."
- "Play podcasts."
- "Enrich my profile."
- "Post this comment."
- "What can others see on my profile?"
