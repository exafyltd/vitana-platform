## What it is

Profile Edit is where the user enriches identity, bio, photo, interests, goals, skills, services, location, languages, and visibility.

Utility screens are everyday tools: assistant chat, calendar, search, profile editing, and public profile preview.

## Why it matters

A richer profile improves Find a Match, business discovery, community trust, and marketplace personalization while preserving privacy boundaries.

## Where to find it

Direct URL: `/profile/edit`.

Sidebar path: **Utility → Profile Edit**.

Module: **Utility**.

## What you see on this screen

- Profile photo, identity, bio, interest, skill, and goal fields
- Business, creator, or service provider fields
- Visibility and privacy controls
- Save and public preview actions

## How to use it

1. Fill in identity and interests first.
2. Add skills and services when the user wants business opportunities.
3. Check visibility before saving sensitive fields.
4. Preview the public profile after major changes.
5. Use the profile to improve matches and clients.

## What you can ask Vitana

- "Enrich my profile."
- "Add this to my bio."
- "Update my interests."
- "Preview my profile."

## Guidance for Vitana Assistant

Utility screens are everyday tools: assistant chat, calendar, search, profile editing, and public profile preview. Explain the screen in plain language first, then offer the next safe action. Ask for confirmation before sending messages, making purchases, booking services, changing privacy, sharing data, altering subscriptions, or deleting anything.
