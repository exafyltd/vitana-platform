## What it is

Public Profile is the Public Profile screen in the Utility module. It gives the user a focused place to understand the current state, choose the next step, and ask Vitana for help.

Utility screens are everyday tools: assistant chat, calendar, search, profile editing, and public profile preview.

4. Return to the main module after the tool action is complete.
