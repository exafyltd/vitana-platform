Presence Debug Panel is the Presence Debug Panel screen in the Overlays module. It gives the user a focused place to understand the current state, choose the next step, and ask Vitana for help.

Overlays are quick action layers that open on top of the current screen so the user can inspect, decide, and continue without losing context.
