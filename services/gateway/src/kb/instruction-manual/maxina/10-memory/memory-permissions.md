## What it is

The **Permissions** screen is part of the Memory module — your timeline, your diary, your recall search, and the permissions that gate it all. In the navigation it sits at Memory → Permissions, and Direct URL: `/memory/permissions`. This chapter explains what the screen contains, why a Maxina community user would open it, what they will see when they do, and how to act on what is there.

## Why it matters

Memory is what makes Vitana feel like it actually knows you across months and devices; the Memory module is where you see, search, and curate what is stored. Every Maxina member arrives at this screen at some point in their first 30 days — sometimes via the Did You Know guided tour, sometimes by tapping a card on Home, sometimes by asking ORB "show me the permissions". Knowing why the screen exists is what stops it from feeling like noise the next time you land here.

## Where to find it

Direct URL: `/memory/permissions`. It lives under the **Memory** module of the sidebar, on the tab labelled **Permissions**. If you ask ORB "open the permissions" the Navigator will route you straight here.

## How to use it

1. Open the screen via the sidebar (Memory → Permissions) or by asking ORB "open permissions".
2. Open Memory to recall something specific, to add a memory by hand, or to audit what Vitana has captured about you.
3. If you are not sure what something on the screen means, ask ORB "what is this card?" — Vitana will read the relevant chapter section aloud.
4. To leave the screen, use the back button or open another sidebar item; nothing on this screen requires you to "save and exit" — your state is persisted automatically.
5. Many screens in the Memory module pair with a related screen: see the related-screens list below for the next logical place to look.
