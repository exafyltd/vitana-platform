### 4. Cloud Run VTID Guard Script ‚úÖ
**File:** `scripts/ensure-vtid.sh`

**Usage:**
```bash

# Example:
./scripts/ensure-vtid.sh vitana-gateway DEV-CICDL-0031 CICDL GATEWAY
```

**Features:**

- Exports `$VTID_LABEL_FLAGS` for use in scripts

```bash

  --region us-central1 \
```
