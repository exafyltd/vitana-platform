# Summary: Community Guide

> A structured overview of the VITANA Community Guide, which serves as the primary knowledge base for AI-assisted user support across all community features.

## Content

### Purpose
The Community Guide helps users navigate all community features and serves as the AI assistant's reference for answering user questions about community functionality.

### Community Sections Covered

**Community Overview (COMM-001)** at `/comm`: Central hub showing activity highlights, quick actions, Vitana Index card, Autopilot card, and navigation tabs.

**Events & Meetups (COMM-002)** at `/comm/events-meetups`: Discover, create, and manage gatherings. Supports time-based filtering (Today, This Week, This Month, All), event cards with images, RSVP/ticketing, event creation wizard, and meetup quick-create. Ticketed events use Stripe for payment.

**Live Rooms (COMM-003)** at `/comm/live-rooms`: Real-time video/audio sessions. Features room status indicators (Live Now, Starting Soon, Scheduled), participant counts, category filtering (Wellness, Fitness, Nutrition, Social), room creation, and notification subscriptions. Hosts can enable recording.

**Media Hub (COMM-004)** at `/comm/media-hub`: Entertainment and education center. Contains Shorts (vertical TikTok-style videos), Podcasts, Music, and Bookmarks. Supports density control (cozy, compact, gallery) and background audio playback.

**Feed (COMM-007)** at `/comm/feed`: Social timeline with Following and Recommended (AI-curated) tabs. Users can create posts with text/images/videos, like, comment, share, and report content. Supports hashtags.

**Challenges (COMM-008)** at `/comm/challenges`: Time-limited community activities for healthy habits. Includes Active, Upcoming, and Completed views with progress tracking, leaderboards, and credit/badge rewards.

**Groups (COMM-009)** at `/comm/groups`: Interest-based micro-communities. My Groups and Recommended (AI-suggested) tabs. Categories: Fitness, Nutrition, Mental Health, Social. Supports public and private groups.

**Matchmaking (COMM-010)** at `/comm/matchmaking`: AI-powered compatibility matching across People, Groups, Coaches, Events, and Analysis tabs. Match scores based on shared interests, goals, activity patterns. Filterable by location and interests.

**Group Detail (COMM-006)** at `/comm/groups/:groupId`: Inside a group view with Posts, Members, Events, About tabs. Admin capabilities include editing info, managing members, moderating posts, creating group events, and setting rules.

### Key Concepts (Glossary)
- RSVP: Free event attendance confirmation
- Meetup: Casual, no-ticket gathering
- Shorts: Vertical short-form videos
- Challenge: Time-limited goal-based activity
- Match Score: AI compatibility percentage
- Autopilot: AI suggestion system
- Vitana Index: Personal wellness score

### Navigation Quick Reference
Eight primary community routes from `/comm` (Overview) through `/comm/matchmaking`, each accessible via Community sidebar and top navigation tabs.
