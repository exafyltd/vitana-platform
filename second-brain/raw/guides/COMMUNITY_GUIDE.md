## Community Overview (COMM-001)

**Route**: `/comm`  
**Also Known As**: Community Home, Community Dashboard

### Overview
The Community Overview is your central hub for all community activity. It displays highlights, quick actions, and a snapshot of what's happening across all community features.

### Getting There
- Click **Community** in the left sidebar
- Or navigate directly to `/comm`

### Features

| Feature | Description |
|---------|-------------|
| **Activity Highlights** | See recent posts, events, and community updates at a glance |
| **Quick Actions** | Fast access to create posts, join events, or find groups |
| **Vitana Index Card** | Your personal health score displayed for quick reference |
| **Autopilot Card** | AI suggestions for community engagement actions |
| **Navigation Tabs** | Switch between Feed, Events, Live Rooms, Groups, and more |

### Actions You Can Take
- **View Autopilot Suggestions**: Click the Autopilot card to see AI-recommended actions
- **Check Vitana Index**: Click the index score to go to your Health dashboard
- **Navigate to Sub-sections**: Use the top navigation to explore specific areas
- **Search**: Use the search button to find people, groups, or events

### Tips & Best Practices
- Check the Autopilot card daily for personalized engagement suggestions
- The activity highlights update in real-time—refresh to see the latest

### Related Screens
- [Feed](#feed-comm-007) for social content
- [Events & Meetups](#events--meetups-comm-002) for gatherings
- [Groups](#groups-comm-009) for communities

## Events & Meetups (COMM-002)

**Route**: `/comm/events-meetups`  
**Also Known As**: Events, Gatherings, Community Calendar

### Overview
Events & Meetups is where you discover, create, and manage community gatherings. From large ticketed events to casual local meetups, everything is organized here.

### Getting There
- Go to **Community** → Click **Events & Meetups** tab
- Or navigate directly to `/comm/events-meetups`

### Features

| Feature | Description |
|---------|-------------|
| **Event Tabs** | Filter by Today, This Week, This Month, or view All Events |
| **Event Cards** | Visual cards showing event image, title, date, location, and attendee count |
| **Create Event** | Launch the event creation wizard |
| **Create Meetup** | Quick-create casual, location-based meetups |
| **RSVP/Buy Tickets** | Join free events or purchase tickets for paid ones |
| **Event Details Drawer** | Slide-out panel with full event information |

### Actions You Can Take
- **Browse Events**: Scroll through event cards or use time filters
- **Create an Event**: Click the "+" button → Select "Event" → Fill in details
- **Create a Meetup**: Click the "+" button → Select "Meetup" → Set location and time
- **RSVP to Event**: Click an event card → Click "RSVP" or "Buy Ticket"
- **Share Event**: Use the share button to send event links via social media or messaging
- **Edit Your Events**: If you're the organizer, click edit on your event card
- **Delete Your Events**: Organizers can delete events they created

### Tips & Best Practices
- Use "This Week" filter to find upcoming events you can still join
- Meetups are great for spontaneous, local gatherings—no tickets required
- Add events to your calendar using the calendar sync button
- Enable notifications to get reminders before events you've RSVP'd to

### Common Questions

**Q: How do I create a ticketed event?**  
A: Click "+" → Select "Event" → Enable "Paid Event" → Set ticket price and quantity.

**Q: Can I sell tickets for someone else's event?**  
A: Yes, if the organizer enabled reselling. Check "Available to Sell" in Business Hub.

**Q: How do I cancel my RSVP?**  
A: Open the event details → Click "Cancel RSVP" or manage in My Tickets.

### Related Screens
- [My Tickets](/my-tickets) for purchased tickets
- [Business Hub](/business) for event earnings
- [Calendar](/calendar) for scheduled events

## Live Rooms (COMM-003)

**Route**: `/comm/live-rooms`  
**Also Known As**: Live Streams, Community Streams, Live Sessions

### Overview
Live Rooms lets you join or host real-time video/audio sessions with the community. Watch live streams, participate in discussions, or broadcast your own content.

### Getting There
- Go to **Community** → Click **Live Rooms** tab
- Or navigate directly to `/comm/live-rooms`

### Features

| Feature | Description |
|---------|-------------|
| **Live Room Cards** | Browse active and scheduled live rooms |
| **Room Status** | See if a room is Live Now, Starting Soon, or Scheduled |
| **Participant Count** | View how many people are in each room |
| **Room Categories** | Filter by topic: Wellness, Fitness, Nutrition, Social, etc. |
| **Create Room** | Start your own live session |
| **Notifications** | Get notified when favorite hosts go live |

### Actions You Can Take
- **Join a Live Room**: Click on any "Live Now" room card to enter
- **Watch as Viewer**: Join without camera/mic to observe
- **Participate**: Request to speak or turn on your camera
- **React**: Send emoji reactions during the stream
- **Chat**: Use the live chat panel to send messages
- **Schedule a Room**: Plan future live sessions
- **Set Notifications**: Bell icon to get alerts when a room goes live
- **Share Room**: Invite others via link

### Tips & Best Practices
- Rooms marked "Starting Soon" let you set a reminder
- If you're hosting, test your audio/video before going live
- Use reactions to engage without interrupting the speaker
- Check the schedule for regular recurring sessions

### Common Questions

**Q: How do I host my own live room?**  
A: Click "Create Room" → Set title, description, and schedule → Click "Go Live" or "Schedule."

**Q: Can I save/record a live room?**  
A: Hosts can enable recording. Recordings appear in Media Hub.

**Q: Why can't I hear anything?**  
A: Check your device volume and ensure the browser has audio permissions.

### Related Screens
- [Media Hub](#media-hub-comm-004) for recorded sessions
- [Groups](#groups-comm-009) for group-specific live rooms

## Media Hub (COMM-004)

**Route**: `/comm/media-hub`  
**Also Known As**: Videos, Podcasts, Music, Shorts

### Overview
The Media Hub is your entertainment and education center. Browse short-form videos (Shorts), listen to podcasts, enjoy music, and access your saved content.

### Getting There
- Go to **Community** → Click **Media Hub** tab
- Or navigate directly to `/comm/media-hub`

### Features

| Feature | Description |
|---------|-------------|
| **Shorts** | Vertical short-form videos (TikTok-style) |
| **Podcasts** | Audio episodes on wellness, longevity, and lifestyle |
| **Music** | Curated wellness and meditation music |
| **Bookmarks** | Your saved media content |
| **Density Control** | Switch between cozy, compact, or gallery view |
| **Search** | Find specific content by title or creator |

### Actions You Can Take
- **Watch Shorts**: Swipe vertically through short videos
- **Like/Share**: Heart a video or share it externally
- **Bookmark**: Save content for later viewing
- **Play Podcast**: Tap to play, control playback
- **Download for Offline**: Save podcasts for offline listening (if available)
- **Switch Views**: Use density control to change layout
- **Search Content**: Find specific media by keyword

### Tips & Best Practices
- Use Bookmarks to build a personal playlist of favorite content
- Shorts are great for quick inspiration during breaks
- Podcasts work well during commutes or workouts
- The Music section has focus and meditation tracks

### Common Questions

**Q: How do I create and upload my own Short?**  
A: Go to Feed → Click "+" → Select "Video" → Record or upload → Add title and tags.

**Q: Can I listen to podcasts in the background?**  
A: Yes, audio continues playing when you navigate to other screens.

**Q: Where are my liked videos?**  
A: Check the Bookmarks tab for all your saved content.

### Related Screens
- [Feed](#feed-comm-007) for posting content
- [Live Rooms](#live-rooms-comm-003) for live video content

## Feed (COMM-007)

**Route**: `/comm/feed`  
**Also Known As**: Timeline, Social Feed, Posts

### Overview
The Feed is your social timeline where you see posts from people you follow and discover recommended content. Share updates, photos, and engage with the community.

### Getting There
- Go to **Community** → Click **Feed** tab
- Or navigate directly to `/comm/feed`

### Features

| Feature | Description |
|---------|-------------|
| **Following Tab** | Posts from people and groups you follow |
| **Recommended Tab** | AI-curated content based on your interests |
| **Create Post** | Share text, images, or videos |
| **Like & Comment** | Engage with posts |
| **Share** | Repost or share externally |
| **Post Cards** | Rich media posts with author info |

### Actions You Can Take
- **Switch Tabs**: Toggle between Following and Recommended
- **Create a Post**: Click "+" → Write text, add media → Post
- **Like a Post**: Tap the heart icon
- **Comment**: Tap comment icon → Write your response
- **Share a Post**: Tap share → Choose destination
- **Follow/Unfollow**: From any user's profile
- **Report Content**: Flag inappropriate posts

### Tips & Best Practices
- The Recommended tab helps you discover new connections
- Use hashtags in your posts to increase visibility
- Engage with comments to build community relationships
- Posts with images get more engagement

### Common Questions

**Q: Why don't I see posts from someone I follow?**  
A: Check if they've posted recently. Also verify you're on the "Following" tab.

**Q: Can I edit a post after publishing?**  
A: Yes, tap the three-dot menu on your post → Edit.

**Q: How do I delete a post?**  
A: Tap the three-dot menu on your post → Delete.

### Related Screens
- [Groups](#groups-comm-009) for group-specific feeds
- [Profile](/profile) to manage your posts

## Challenges (COMM-008)

**Route**: `/comm/challenges`  
**Also Known As**: Community Challenges, Wellness Challenges

### Overview
Challenges are time-limited community activities designed to motivate healthy habits. Join challenges, track your progress, and compete with others.

### Getting There
- Go to **Community** → Click **Challenges** tab
- Or navigate directly to `/comm/challenges`

### Features

| Feature | Description |
|---------|-------------|
| **Active Challenges** | Challenges you've joined that are currently running |
| **Upcoming Challenges** | Future challenges you can join early |
| **Completed Challenges** | Your challenge history and achievements |
| **Progress Tracking** | See your daily/weekly progress |
| **Leaderboard** | Compare your progress with others |
| **Rewards** | Earn badges and credits for completing challenges |

### Actions You Can Take
- **Browse Challenges**: Scroll through available challenges
- **Join a Challenge**: Click "Join" on any challenge card
- **Log Progress**: Update your daily activity within a challenge
- **View Leaderboard**: See how you rank against other participants
- **Share Achievements**: Post your milestones to the Feed
- **Leave Challenge**: Exit a challenge you no longer want to participate in

### Tips & Best Practices
- Join challenges that align with your health goals
- Set daily reminders to log your progress
- Challenges with friends are more motivating—invite them!
- Completing challenges earns you Vitana credits

### Common Questions

**Q: How do I track my progress?**  
A: Open an active challenge → Tap "Log Progress" → Enter your activity.

**Q: What happens if I miss a day?**  
A: Most challenges allow catch-up, but check the specific challenge rules.

**Q: How do I earn rewards?**  
A: Complete challenge milestones to earn badges and credits automatically.

### Related Screens
- [Health](/health) for tracking wellness metrics
- [Wallet](/wallet) for viewing earned credits

## Groups (COMM-009)

**Route**: `/comm/groups`  
**Also Known As**: Community Groups, Interest Groups

### Overview
Groups bring together people with shared interests. Join existing groups or create your own to build micro-communities within VITANA.

### Getting There
- Go to **Community** → Click **Groups** tab
- Or navigate directly to `/comm/groups`

### Features

| Feature | Description |
|---------|-------------|
| **My Groups Tab** | Groups you've joined |
| **Recommended Tab** | AI-suggested groups based on your interests |
| **Group Cards** | Visual preview with name, members, and description |
| **Create Group** | Start your own community group |
| **Search Groups** | Find groups by name or topic |
| **Group Categories** | Filter by Fitness, Nutrition, Mental Health, Social, etc. |

### Actions You Can Take
- **Browse Groups**: Scroll through group cards
- **Join a Group**: Click "Join" on any group card
- **Create a Group**: Click "+" → Fill in name, description, rules
- **Search**: Use the search bar to find specific groups
- **Filter by Category**: Narrow down groups by interest area
- **View Group Details**: Click a card to see full info before joining

### Tips & Best Practices
- Join groups aligned with your wellness goals
- Active participation builds your community reputation
- As a group admin, set clear rules and moderate regularly
- Private groups require approval; public groups are open to all

### Common Questions

**Q: How do I leave a group?**  
A: Open the group → Settings (gear icon) → "Leave Group."

**Q: Can I make my group private?**  
A: Yes, when creating or editing, toggle "Private Group" on.

**Q: How do I invite people to my group?**  
A: Open group → "Invite Members" → Share the invite link.

## Matchmaking (COMM-010)

**Route**: `/comm/matchmaking`  
**Also Known As**: Find Connections, People Matching, Discovery

### Overview
Matchmaking uses AI to help you find compatible people, groups, coaches, and events based on your interests, goals, and activity.

### Getting There
- Go to **Community** → Click **Matchmaking** tab
- Or navigate directly to `/comm/matchmaking`

### Features

| Feature | Description |
|---------|-------------|
| **People Tab** | Find compatible community members |
| **Groups Tab** | Discover groups that match your interests |
| **Coaches Tab** | Find wellness coaches and professionals |
| **Events Tab** | Events recommended based on your profile |
| **Analysis Tab** | AI-powered compatibility insights |
| **Filters** | Refine matches by location, interests, goals |
| **Match Score** | Compatibility percentage with each match |

### Actions You Can Take
- **Browse Matches**: Scroll through match cards in each tab
- **Apply Filters**: Click filter icon to narrow results
- **View Profile**: Tap a match card to see full profile
- **Connect**: Send a connection request to people
- **Join Group/Event**: Quick-join from match cards
- **View Compatibility Analysis**: See why you were matched

### Tips & Best Practices
- Complete your profile for better match accuracy
- The Analysis tab explains your compatibility scores
- Use filters to find local connections
- Higher match scores indicate stronger alignment

### Common Questions

**Q: How is my match score calculated?**  
A: It considers shared interests, goals, activity patterns, and engagement style.

**Q: Can I hide from matchmaking?**  
A: Yes, go to Settings → Privacy → Toggle "Show in Matchmaking" off.

**Q: Why aren't my matches accurate?**  
A: Update your profile and interests for better recommendations.

### Related Screens
- [Groups](#groups-comm-009) to join matched groups
- [Events & Meetups](#events--meetups-comm-002) for matched events
- [Profile](/profile) to improve your match quality

## Group Detail (COMM-006)

**Route**: `/comm/groups/:groupId`  
**Also Known As**: Inside a Group, Group Page

### Overview
The Group Detail screen is what you see when you enter a specific group. It shows the group's posts, members, events, and settings.

### Getting There
- Go to **Community** → **Groups** → Click on any group card
- Or click a group link from anywhere in the app

### Features

| Feature | Description |
|---------|-------------|
| **Group Header** | Name, cover image, member count, description |
| **Posts Tab** | Group-specific posts and discussions |
| **Members Tab** | List of all group members |
| **Events Tab** | Events organized by or for this group |
| **About Tab** | Group rules, admins, and details |
| **Group Chat** | Real-time messaging with members |
| **Create Post** | Share content within the group |

### Actions You Can Take
- **Post in Group**: Create text, image, or video posts
- **Comment & Like**: Engage with group posts
- **View Members**: See who's in the group
- **Start Group Chat**: Message group members
- **RSVP to Group Events**: Join events organized by the group
- **Invite Members**: Share group invite link
- **Leave Group**: Exit via settings
- **Report Group**: Flag inappropriate group content

### Admin Actions (if you're an admin)
- **Edit Group Info**: Update name, description, cover image
- **Manage Members**: Approve requests, remove members
- **Moderate Posts**: Delete inappropriate content
- **Create Group Events**: Organize events for members
- **Set Group Rules**: Define community guidelines

### Tips & Best Practices
- Check the About tab to understand group rules before posting
- Use @ mentions to notify specific members
- Group events are a great way to meet members in person
- Private groups keep discussions confidential

### Common Questions

**Q: How do I become a group admin?**  
A: The group creator can promote you. Ask them directly.

**Q: Can I mute group notifications?**  
A: Yes, tap the bell icon in the group header to mute.

**Q: How do I report a member?**  
A: Go to their profile → Three-dot menu → "Report User."

### Related Screens
- [Groups](#groups-comm-009) to find more groups
- [Events & Meetups](#events--meetups-comm-002) for group events
