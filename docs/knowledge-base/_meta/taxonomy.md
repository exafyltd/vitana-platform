# Knowledge Base Taxonomy & ID Scheme

## ID Format

```
kb-{lang}-{section}-{article}
```

Examples:
- `kb-en-01-01` → English, Section 01 (Foundation), Article 01 (What Is Maxina?)
- `kb-de-01-01` → German, Section 01, Article 01
- `kb-en-12-07` → English, Section 12 (Health Tracking), Article 07 (Biomarkers)
